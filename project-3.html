Project Overview

This web based project was developed as a proof of concept for the Oil and Gas Industry. The project utilited an algortithm that aided in the prediction of historical oil and gas dataset.

  • Developed a database management system where Information can be stored and retrieved.
  • Create the backend using PHP in other to send request to the database as well as retrieve data.
  • Create a frontend utilizing the web, using HTML, CSS and JavaScript.
  • Integrate D3.js Chart JavaScript library for visualisation of the results.
